# Python has the following built-in data types, grouped into categories:

- **Text Type**: `str`

- **Numeric Types**: `int`, `float`, `complex`

- **Sequence Types**: `list`, `tuple`, `range`

- **Mapping Type**: `dict`

- **Set Types**: `set`, `frozenset`

- **Boolean Type**: `bool`

- **Binary Types**: `bytes`, `bytearray`, `memoryview`

- **None Type**: `NoneType`
